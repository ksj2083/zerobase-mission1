[기능 설명]
1. 와이파이 정보 가져오기 - openAPI에서 서울시 공공 와이파이 정보를 받아서 DB에 저장
2. 근처 와이파이 검색 - 입력한 위도, 경도를 기준으로 가까운 순서로 인근 공공와이파이를 출력
3. 히스토리 페이지로 이동 - 검색기록을 보여주는 페이지로 이동
